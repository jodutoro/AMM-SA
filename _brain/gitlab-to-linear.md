The SearchAtlas engineering and ops team has fully migrated from GitLab to Linear for all issue tracking.

**Why:** GitLab was the legacy tracker; Linear is now the single source of truth for all tickets, bugs, and project work.

**How to apply:** Always use Linear (via MCP tools or the Linear web app) for issue creation, searching, and status updates. Never suggest GitLab for any ticket-related task. When referencing issue tracking tools, default to Linear.
