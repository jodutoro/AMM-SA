name: Avoid same image repeating across page sections
description: Never use the same product image in category cards, featured products, AND catalog — it looks bad when stacked vertically
type: feedback
---

Never repeat the same image across multiple stacked sections on the same page.

**Why:** On Bella Caramella, blusa-07 (lace corset) appeared in: (1) category card, (2) featured product card, (3) catalog tile — three times in a row as the user scrolled. The user flagged this explicitly as "too much."

**How to apply:** When a page has category cards + featured products + a catalog/gallery section, each section must use a distinct set of images. Assign images to sections in priority order: categories get the most iconic shot per category, featured gets the second-best unique image per category, catalog fills in with the remaining images not used above.
