# AMM Toolkit — What's New (April 2026)

> This is a curated digest for agency partners — filtered and explained in terms of what actually matters for your work.

## Claude Code — v2.1.111 (April 16, 2026)

**What changed:** New `xhigh` effort level for Claude Opus 4.7, tunable via `/effort`, `--effort`, and the model picker. Auto mode is now available for Max subscribers when using Opus 4.7.

**Why it matters for your agency:** `xhigh` sits between `high` and `max` — useful when you want more reasoning than `high` gives you but don't want to burn the full `max` token budget. For the audit + content generation workflows in this toolkit, `xhigh` is often the sweet spot: deep enough to catch nuanced SEO and compliance issues, cheap enough to run across a full client portfolio without hitting usage caps. Auto mode for Max subscribers means the model will adapt effort level per task automatically — let it handle the routing.

**Action needed:** Nothing required. If you are on the Max plan, try `/effort xhigh` on your next audit run and compare output quality. If you are running this toolkit's workflows programmatically, consider adding `--effort xhigh` to your default command for the audit and content-brief steps.

## Claude Code — v2.1.112 (April 16, 2026)

**What changed:** Fixed a bug where "claude-opus-4-7 is temporarily unavailable" would appear under auto mode.

**Why it matters for your agency:** If you were seeing intermittent failures when running auto mode, update — this fix removes the transient unavailability error that was blocking longer sessions.

**Action needed:** Update Claude Code to v2.1.112 or later. On macOS/Linux: `npm install -g @anthropic-ai/claude-code@latest`. On Windows (WSL): same command inside Ubuntu.
